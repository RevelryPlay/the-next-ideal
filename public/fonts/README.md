# Font Files

Add your font files to this directory:

## Required Files

### Brothers OT Bold (for headings)
- `Brothers-OT-Bold.woff2`
- `Brothers-OT-Bold.woff`

### Interstate Light (for body text)
- `Interstate-Light.woff2`
- `Interstate-Light.woff`

## Converting Fonts

If you have .otf or .ttf files, convert them to .woff2 and .woff using:
- https://cloudconvert.com/otf-to-woff2
- https://www.fontsquirrel.com/tools/webfont-generator

## Note

These are commercial fonts. Make sure you have the appropriate license for web use.
